# 🎨 Meu Projeto Individual – Blog de Arte

Este repositório contém o desenvolvimento do meu projeto individual acadêmico, com o objetivo de crar um **blog pessoal sobre arte**, com foco nas minhas experiências com **pintura em aquarela e guache**.

A minha proposta é compartilhar de forma simples e autêntica, como a arte pode ser acessível a qualquer pessoa, independentemente de técnica ou experiência, incentivando outros a experimentarem a criatividade como forma de expressão.

O site contará com:

* Apresentação pessoal e contexto sobre o projeto
* Sistema de cadastro e login de usuários
* Redirecionamento de conteúdo com base no nível de experiência (iniciante ou intermediário)
* Tutoriais básicos para quem deseja começar
* Galeria de pinturas autorais
* Área restrita para usuários cadastrados

Este projeto está sendo desenvolvido com:

* HTML, CSS e JavaScript
* Node.js (integração com API)
* Implementação de gráficos utilizando Chart.js
* Banco de Dados

Também contará com:
* Documentação completa do Projeto
* Utilização de ferramentas de gestão para organização

📌 **Observação:**
Este projeto é desenvolvido exclusivamente para fins acadêmicos, como parte do curso de Sistemas de Informação.

🚧 O repositório será atualizado continuamente conforme o avanço do desenvolvimento do projeto.
